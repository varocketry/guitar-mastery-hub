## üìñ THEORY: Three-Note Power Chords

### Beyond the Two-Note Shape

**So far you've learned**: Root + Fifth (two notes)

**Today you'll add**: The octave (three notes)

**Why add the octave?**
- ‚úÖ Fuller, richer sound
- ‚úÖ More presence in mix
- ‚úÖ Better for rhythm parts
- ‚úÖ Professional voicing
- ‚úÖ Classic rock sound

### The Three-Note Shape

**E5 Three-Note Voicing**:
```
e|---
B|---
G|---
D|--2‚Üê Fifth (B note)
A|--2‚Üê Root (E note)
E|--0‚Üê Root octave (E note - lower)
     ‚Üë
  All three notes are part of E5!
```

**What changed?** Added the octave root on 4th string (D string)

### Interval Analysis

**The three notes**:
1. **Root** (E on 6th string)
2. **Fifth** (B on 5th string - 7 semitones above root)
3. **Octave** (E on 4th string - 12 semitones above root)

**Why this sounds good**:
- Octaves reinforce the root (same note, higher pitch)
- Perfect fifths are consonant (stable, pleasing)
- No dissonance (no clashing notes)

### Two-Note vs. Three-Note Comparison

**Two-Note (Basic)**:
```
E5 (two notes):
D|--2--
A|--2--
E|--0--

Sound: Strong, focused, tight
Best for: Fast passages, metal, precise rhythm
```

**Three-Note (Full)**:
```
E5 (three notes):
D|--2--
A|--2--
E|--0--
G|--0-- ‚Üê Added octave

Sound: Full, rich, powerful
Best for: Rhythm parts, classic rock, fuller arrangements
```

**Both are correct!** Use situationally.

**Method 1: Index-Ring-Pinky** (Recommended)
```

```

E|--1‚Üê Index finger (barre)
```

**Which to use?**
- Method 1 (with pinky) allows more speed
- Method 2 (without pinky) if you have weak pinky

### Step-by-Step: Learning Three-Note Shapes

**Learning E5 (Three-Note)** (15 minutes)

**Step 1: Add the Octave** (5 min)
1. Form regular two-note E5 (strings 6 and 5)
2. Add pinky on D string, 2nd fret
3. Strum all three strings (E-A-D)
4. Listen to the fuller sound
5. Repeat 20 times

**Step 2: Comparison Exercise** (5 min)
```
Two-note E5: Strum 4 times
Three-note E5: Strum 4 times

Hear the difference?
Three-note has more "body" and presence
```

**Step 3: Clean Sound Check** (5 min)
- All three strings must ring clearly
- If buzzing: Check pinky angle and pressure
- If muted: Check fingers aren't touching adjacent strings
- Practice until consistent

**F5 Three-Note**:
```

B|---
G|---

## üé∏ TECHNIQUE #2: Riff Construction Principles

### What Makes a Great Riff?

**A riff** = Short, repeated musical phrase that defines a song

**Characteristics of memorable riffs**:
- ‚úÖ Repetitive (easy to remember)
- ‚úÖ Rhythmically interesting (not just straight quarter notes)
- ‚úÖ Melodically simple (usually 3-6 notes)
- ‚úÖ Powerful (uses power chords or strong intervals)
- ‚úÖ Catchy (gets stuck in your head)

**Examples**:
- "Smoke on the Water" - Deep Purple
- "Iron Man" - Black Sabbath
- "Seven Nation Army" - White Stripes
- "Smells Like Teen Spirit" - Nirvana

### Riff Construction Formula

**Basic riff building blocks**:

**1. Root Movement** (where the bass note goes)
```
Example: E ‚Üí G ‚Üí A
Creates tension and resolution
```

**2. Rhythm Pattern** (when you play the notes)
```
Example: Short-short-long
Eighth-eighth-quarter note
```

**3. Repetition** (how often it repeats)
```
Usually: 2, 4, or 8 measure phrases
Odd numbers (3, 5, 7) create tension
```

**4. Variation** (how it changes)
```
Play main riff 3 times
Play variation on 4th time
This is VERY common!
```

### The "Call and Response" Technique

**Pattern**: Statement ‚Üí Answer ‚Üí Statement ‚Üí Answer

**Example**:
```
Measure 1-2: E5-G5-A5 (statement)
Measure 3-4: A5-G5-E5 (answer - reverses the statement)

This creates satisfying musical conversation!
```

### Creating Your First Riff

**Step-by-Step Riff Creation** (20 minutes):

**Step 1: Choose 3-4 Power Chords** (5 min)
- Pick chords that sound good together
- Example: E5, G5, A5, D5
- Play them in different orders
- Which order sounds best?

**Step 2: Add a Rhythm** (5 min)
```
Try these patterns with your chord choices:

Pattern A: / / / /     (all quarter notes)
Pattern B: / / . /     (quarter-quarter-rest-quarter)
Pattern C: / . / / .   (quarter-rest-quarter-quarter-rest)

Which creates most interest?
```

**Step 3: Create a Two-Measure Phrase** (5 min)
- Combine chords and rhythm
- Play it 10 times
- Adjust what doesn't sound good
- Simplify if too complex

**Step 4: Add Variation** (5 min)
- Play main phrase 3 times
- Change something on 4th time
- Maybe: different ending chord
- Or: different rhythm
- Or: add a pause

**Congratulations - you've written a riff!**

### Practice Routine: Riff Creation

**Exercise 1: Riff Deconstruction** (15 minutes)
```
Take "Smoke on the Water" opening:
G5-Bb5-C5  G5-Bb5-Db5-C5  G5-Bb5-C5-Bb5-G5

Analyze:
- How many different chords? (4: G5, Bb5, C5, Db5)
- What's the rhythm pattern? (Mostly eighth notes)
- How does it create interest? (Ascending then descending motion)
- What makes it memorable? (Simple, repetitive, powerful)

Play it slowly 20 times
```

**Exercise 2: Variation Workshop** (15 minutes)
```
Take simple riff: E5-G5-A5-G5 (repeat)

Create 3 variations:
1. Change rhythm (make one chord longer)
2. Change order (A5-G5-E5-G5)
3. Add pause (E5-G5-[pause]-A5)

Which variation sounds best?
```

**Exercise 3: Original Creation** (20 minutes)
```
Your assignment:
- Choose 3 power chords
- Create 4-measure riff
- Play it 30 times
- Record it
- Listen back critically
- Refine it

This is YOUR signature riff!
```

## üé∏ TECHNIQUE #3: String Dampening & Control

### Selective String Ringing

**Challenge**: When playing power chords, you only want CERTAIN strings to ring.

**Example with E5 three-note**:
```
You WANT these to ring:
D|--2--  ‚úÖ
A|--2--  ‚úÖ
E|--0--  ‚úÖ

You DON'T want these:
G|--x--  ‚ùå (mute this!)
B|--x--  ‚ùå (mute this!)
e|--x--  ‚ùå (mute this!)
```

### The "Choke" Technique

**What it is**: Lifting fingers slightly to stop strings from ringing

**When to use it**:
- Creating rhythmic effects
- Stopping ring between riff phrases
- Staccato (short, punchy) sounds

**How to do it**:
1. Play power chord fully (all notes ring)
2. Slightly release finger pressure (keep contact but don't press)
3. Strings stop ringing but fingers stay in position
4. Can quickly press again for next note

**Practice Exercise** (10 minutes):
```
E5 (play) - choke - E5 (play) - choke - E5 (play) - choke

Rhythm: Quarter note - eighth rest - quarter note - eighth rest

Creates: CHUNG-chuck-CHUNG-chuck-CHUNG-chuck sound
```

### Palm Muting with Three-Note Chords

**Technique**: Same as with two-note chords, but muting THREE strings now

**Position**:
- Side of palm rests on E, A, AND D strings
- Right at the bridge
- Light pressure

**Sound difference**:
- Unmuted three-note: **CHUNG** (open, ringing, loud)
- Palm-muted three-note: **chunk** (tight, controlled, aggressive)

**Practice Pattern** (15 minutes):
```
E5 (open) - E5 (muted) - E5 (open) - E5 (muted)
|    /    |    /      |    /    |    /      |

4 repetitions
Focus on consistent mute pressure
Tempo: 60 BPM
```

## üîß TROUBLESHOOTING

### Problem: Three-Note Power Chords Buzz on D String

**Symptoms**:
- First two strings clean, third string (octave) buzzes
- Pinky note doesn't ring clearly

**Solutions**:
- Pinky probably not pressing hard enough
- Check pinky angle (perpendicular to fretboard)
- Build pinky strength with dedicated exercises
- Position pinky very close to fret wire
- May take weeks for pinky to catch up to other fingers

### Problem: Created Riffs Sound Generic

**Symptoms**:
- Riffs work technically but lack personality
- Sound like "every other rock song"

**Solutions**:
- Add rhythmic variation (not all straight notes)
- Use unexpected shape changes
- Add pauses (silence is powerful!)
- Listen to diverse rock styles for inspiration
- Don't judge early - create many, keep best

### Problem: Can't Maintain Consistent Palm Muting While Playing Riffs

**Symptoms**:
- Palm mute position shifts
- Some notes muted, some open (unintentionally)

**Solutions**:
- This is THE challenge of riff playing!
- Practice palm mute position without playing first
- Start very slow (40 BPM)
- Mark palm position with visual cue
- Build consistency over weeks
- Even slight inconsistency improves with practice

### Problem: Fingers Get Tangled on Three-Note Chords

**Symptoms**:
- Fingers hit each other
- Can't get all three in position
- Uncomfortable hand position

**Solutions**:
- Check finger order (index-ring-pinky most efficient)
- Ensure fingers approach from above (not sliding in)
- Practice each finger position separately
- Might need to adjust thumb position
- Film your hand to spot issues
