## üéµ SECTION 1: BUILDING TRIADS

### What Is a Triad?

**Triad:** A three-note chord built by stacking thirds.

**Formula:** Root + 3rd + 5th (from a scale)

**Example: C Major Triad**
```
C Major Scale: C - D - E - F - G - A - B
Pick every other note: C (1) - E (3) - G (5)
Result: C Major Chord
```

**This is how ALL triads are built.**

---

### The Four Triad Types

**1. Major Triad**
- **Formula:** 1 - 3 - 5 (from major scale)
- **Sound:** Happy, bright, stable
- **Example:** C Major = C - E - G
- **Half steps:** Root, +4 half steps, +3 half steps

**2. Minor Triad**
- **Formula:** 1 - ‚ô≠3 - 5 (flatted 3rd)
- **Sound:** Sad, dark, stable
- **Example:** C Minor = C - E‚ô≠ - G
- **Half steps:** Root, +3 half steps, +4 half steps

**3. Diminished Triad**
- **Formula:** 1 - ‚ô≠3 - ‚ô≠5 (flatted 3rd and 5th)
- **Sound:** Tense, unstable, unresolved
- **Example:** C Diminished = C - E‚ô≠ - G‚ô≠
- **Half steps:** Root, +3 half steps, +3 half steps

**4. Augmented Triad**
- **Formula:** 1 - 3 - #5 (sharped 5th)
- **Sound:** Bright, tense, unusual
- **Example:** C Augmented = C - E - G#
- **Half steps:** Root, +4 half steps, +4 half steps

---

### The Key Difference: Major vs. Minor

**The only difference is ONE NOTE‚Äîthe 3rd.**

**C Major:** C - E - G (major 3rd = 4 half steps from root)  
**C Minor:** C - E‚ô≠ - G (minor 3rd = 3 half steps from root)

**That one half-step changes the entire emotional character.**

## üéµ SECTION 2: SEVENTH CHORDS

### Adding the Seventh

**Seventh chords = Triad + one more note (the 7th scale degree)**

**Four Common Types:**

**1. Major 7th (maj7)**
- **Formula:** 1 - 3 - 5 - 7 (all from major scale)
- **Sound:** Dreamy, sophisticated, jazz
- **Example:** Cmaj7 = C - E - G - B
- **Chords:** Cmaj7, Gmaj7, Fmaj7

**2. Dominant 7th (7)**
- **Formula:** 1 - 3 - 5 - ‚ô≠7 (major triad + flatted 7th)
- **Sound:** Tense, wants resolution, bluesy
- **Example:** C7 = C - E - G - B‚ô≠
- **Usage:** Creates tension, resolves to tonic
- **Chords:** G7, D7, A7, E7

**3. Minor 7th (m7)**
- **Formula:** 1 - ‚ô≠3 - 5 - ‚ô≠7 (minor triad + flatted 7th)
- **Sound:** Mellow, jazzy, smooth
- **Example:** Cm7 = C - E‚ô≠ - G - B‚ô≠
- **Chords:** Am7, Em7, Dm7

**4. Minor 7‚ô≠5 (Half-Diminished)**
- **Formula:** 1 - ‚ô≠3 - ‚ô≠5 - ‚ô≠7 (diminished triad + flatted 7th)
- **Sound:** Very tense, jazzy

- **Usage:** Jazz, leading to minor chords

## üéµ SECTION 3: HARMONIC FUNCTION

### The Three Functions

**All chords in a key serve one of three functions:**

**1. TONIC Function (Home)**
- **Chords:** I, vi (iii in some contexts)
- **Feel:** Stable, resolved, restful
- **Job:** Establish the key, provide resolution
- **Example in C Major:** C major, A minor

**2. SUBDOMINANT Function (Away From Home)**
- **Chords:** IV, ii
- **Feel:** Moving away, building anticipation
- **Job:** Set up dominant or create contrast
- **Example in C Major:** F major, D minor

**3. DOMINANT Function (Tension, Wants to Resolve)**
- **Chords:** V, vii¬∞
- **Feel:** Tense, unstable, pulling toward tonic
- **Job:** Create tension that resolves to I
- **Example in C Major:** G major, B diminished

---

### Why I-IV-V Works

**The most common progression in Western music:**

**I (Tonic) ‚Üí IV (Subdominant) ‚Üí V (Dominant) ‚Üí I (Tonic)**

**In C Major: C ‚Üí F ‚Üí G ‚Üí C**

**Why it works:**
1. **I = Home** (established)
2. **IV = Movement** (away from home)
3. **V = Tension** (maximum pull back to home)
4. **I = Resolution** (satisfying return)

**This is the foundation of countless songs.**

---

### Chord Progressions in Major Keys

**Common progressions (using Roman numerals):**

**1. I - V - vi - IV** ("Axis of Awesome" progression)
- Used in thousands of pop songs
- **Example:** C - G - Am - F
- **Function:** Tonic ‚Üí Tension ‚Üí Contrast ‚Üí Pre-dominant ‚Üí repeat

**2. I - IV - V - IV** (Classic rock)
- Simple, powerful, memorable
- **Example:** C - F - G - F
- **Function:** Home ‚Üí Away ‚Üí Tension ‚Üí Away ‚Üí repeat

**3. I - vi - IV - V** (50s progression)
- Doo-wop, early rock & roll
- **Example:** C - Am - F - G
- **Function:** Home ‚Üí Relative minor ‚Üí Pre-dominant ‚Üí Tension

**4. ii - V - I** (Jazz/Functional)
- Strong resolution, very satisfying
- **Example:** Dm - G - C
- **Function:** Pre-dominant ‚Üí Dominant ‚Üí Tonic (strongest motion)

## üéµ SECTION 4: DIATONIC CHORDS IN A KEY

### The Seven Chords of Any Major Key

**Every major key contains seven chords built from its scale.**

**C Major Example:**
```
C Major Scale: C - D - E - F - G - A - B

I: C major (C - E - G)
ii: D minor (D - F - A)
iii: E minor (E - G - B)
IV: F major (F - A - C)
V: G major (G - B - D)
vi: A minor (A - C - E)
vii¬∞: B diminished (B - D - F)
```

**Pattern (same in every major key):**
- I = Major
- ii = minor
- iii = minor
- IV = Major
- V = Major
- vi = minor
- vii¬∞ = diminished

**Uppercase Roman numeral = Major chord**  
**Lowercase = minor chord**  
**¬∞ = diminished**

---

### Using Diatonic Chords

**Any progression using these chords will "work" in that key.**

**In C Major, these progressions all work:**
- C - F - G (I - IV - V)
- C - Am - F - G (I - vi - IV - V)
- Dm - G - C (ii - V - I)
- Am - F - C - G (vi - IV - I - V)

**Because all chords come from C major scale.**

## üéµ SECTION 6: ADVANCED CONCEPTS (PREVIEW)

### Chord Substitutions

**You can substitute chords with similar functions:**

**1. Relative Minor Substitution**
- vi can substitute for I (both tonic function)
- Example: Instead of C major, use A minor

**2. Secondary Dominants**
- Borrowing V chords from other keys
- Example: In C major, use A7 (V of D minor) before Dm

**3. Modal Interchange**
- Borrowing chords from parallel minor key
- Example: In C major, use F minor (from C minor)

**These are advanced techniques you can explore later.**

---

### Extended Chords (9ths, 11ths, 13ths)

**9th Chord:** Add the 9th scale degree (1-3-5-7-9)
- **Example:** Cmaj9 = C - E - G - B - D
- **Sound:** Lush, sophisticated

**11th Chord:** Add the 11th (1-3-5-7-9-11)
- **Example:** C11 = C - E - G - B‚ô≠ - D - F
- **Sound:** Complex, jazzy

**13th Chord:** Add the 13th (1-3-5-7-9-11-13)
- **Example:** C13 = C - E - G - B‚ô≠ - D - F - A
- **Sound:** Very full, jazz standard

**These are built the same way‚Äîjust stacking more scale degrees.**
